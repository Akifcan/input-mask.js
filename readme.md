# input mask js
## input mask library for javascript

### example

```
	<input 
		type="text" 
		class="mask"
		data-mask='000-000-00'
	/>

	<input 
		type="text" 
		class="mask"
		data-mask='00:00'
	/>

	<input 
		type="text" 
		class="mask"
		data-mask='####-0'
	/>


```

* input class should be `mask`
